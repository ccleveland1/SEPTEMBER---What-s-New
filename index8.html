What's New
Version History
New features, enhancements and bug fixes.
Version OAN 20-08.0.0
OANet for HRNew Feature
MIS has created a database (9950HR) and rolled out OANet for internal use by HR, including an online application and full set of documents (e-signature & information). This tool will complement HR’s use of ERP and WebRecruiter in managing corporate and field employees.
Payrol Tab EnhancementEnhancement
The Payroll tab has been updated with a new layout – despite the cosmetic changes, the main features available will remain the same. The primary change is to the location of the action buttons; while previously located at the bottom of the screen, the actions available to the user can now be found between the Batch and Timecard grids. With this update, the actions available to the user are more apparent for the selected batch/timecard, and more timecards can be displayed in the second grid.
Time Entry PageEnhancement
On the Time Entry popup, users may now select different Pay Code categories. A separate dropdown icon has been added next to the Pay Code column; by clicking this icon, users can select different Pay Code categories
Time Card Rounding Rule FieldEnhancement
A time card rounding rule field will now be available in the Client/Placement screen. The rounding rule selected will be applied to People Portal Clock In/Out time cards
Scanned Document Audit ReportEnhancement
The Scanned Document Audit Report has been updated. When searching for who might be missing the document in question, the system will now look in all OA databases and see if any file with the matching Social Security number has the document in it. If so, the candidate will not show up on the report as “missing”
ActOne Connection Center Job PortalEnhancement
Resumes will now be extracted from various data sources and consolidated on a specialized database for the ActOne Connection Center search portal.
Federal I-9 E-Signature ProcessEnhancement
As a measure to facilitate remote application, the Federal I-9 form has been converted for E-Signature use. Applicants may now receive and sign the I-9 form electronically; if necessary, certain portions of the form can be sent to a designated representative for completion. In order to smoothly handle this form, the document has been divided into two parts – Page 1 (employee section) and Page 2 (Identity Document verification section).
For more information please visit the Release Notes.
UTC ImplementationBug Fix
The creation and modification of records within OANet will now be logged according to UTC time and converted to local time for display when viewed by a branch.
Manual and File Upload Time Card TypesBug Fix
The default time card type for Manual and File Upload time cards has been updated – these time cards are now automatically assigned TC type “AlternateTC
